# Практика

1. Реализовать контекстное меню. Список хранить в памяти.

2. Применить меню к домашке с лекции №15 (персонаж). Создать actions: Jump, Remove, ChangeColor

3. Меню должно всегда открыватся в окне, не создавая скрола.
